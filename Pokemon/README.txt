Projet Pokémon 

	Description du Jeu :

J'ai fais un jeu Pokémon qui est fortement inspiré de Pokémon Jaune, dans ce jeu vous pourrez combattre avec d'autres Pokémon,
parcourir les différentes routes disponibles. Votre mission sera de combattre des Pokémon sauvages afin d'accomplir les 3 Objectifs.
(La touche permet ENTRER permet d'avancer dans les dialogues)

	Fonctionnalité : 

- 3 Objectifs 
- 3 routes 
- 4 Pokémon sauvages 
- Combat PvE avec des Pokémon généré aléatoire et équilibré 
- Les Pokémon respectent au mieux les statistiques du vrai jeu 
- On peut voir les statistiques de son Pokémon 
- Centre Pokémon pour soigner son Pokémon 
- Carte permettant se déplacer 
- Sac avec un objet la potion (On peut l'ouvrir pendant un combat)
- Gain Expérience aléatoire en fonction du Pokémon combattue 
- Un maximum de bug/failles ont été corrigés
